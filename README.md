# This is the AI code and API for the NeuroSense application

> An application dedicated to predict Multiple sclerosis (MS) using a basic Random Classifier Machine-Learning model trained on local dataset of images, audio, survey questions

## 1. Technology Stack
- Python >= 13
- Basic Machine-Learning libraries(numpy, matplotlib, scikit-learn, pandas)
- Streamlit (for interactive UI tests)
- Flask (for exposed APIs)

## 2. How to run
```sh 
pip install -r requirements.txt
```
